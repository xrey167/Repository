# xAI (Grok)

## Overview
xAI develops Grok, a conversational AI with real-time knowledge access and a distinctive personality. Integrated with X (Twitter) for current information.

## Key Features
- Real-time information access
- X (Twitter) integration
- Long context support
- Vision capabilities
- Unrestricted conversation style
- Reasoning capabilities

## Models

| Model | Description | Context | Best For |
|-------|-------------|---------|----------|
| grok-2 | Latest flagship | 128K | Complex tasks |
| grok-2-mini | Fast and efficient | 128K | Quick responses |
| grok-vision-beta | Multimodal | 8K | Image understanding |

## API Information
- **Base URL**: `https://api.x.ai/v1`
- **Authentication**: Bearer token
- **Format**: OpenAI-compatible

## SDKs & Libraries
- **Official**: OpenAI-compatible clients work
- **Python**: Use `openai` with custom base URL
