# OpenAI

## Overview
OpenAI provides state-of-the-art large language models including GPT-4, GPT-4o, and the o1 reasoning models, plus DALL-E for image generation and Whisper for speech recognition.

## Key Features
- GPT-4 and GPT-4o multimodal models
- o1 reasoning models for complex tasks
- Function calling and structured outputs
- Vision capabilities
- Fine-tuning support
- Assistants API with tools

## Models

| Model | Description | Context | Best For |
|-------|-------------|---------|----------|
| gpt-4o | Fastest GPT-4 class model | 128K | General use, multimodal |
| gpt-4o-mini | Cost-efficient model | 128K | High-volume tasks |
| o1 | Advanced reasoning | 200K | Complex reasoning, math, code |
| o1-mini | Fast reasoning | 128K | STEM tasks |
| gpt-4-turbo | Previous flagship | 128K | Complex tasks |

## API Information
- **Base URL**: `https://api.openai.com/v1`
- **Authentication**: Bearer token (API key)
- **Rate Limits**: Varies by tier (TPM/RPM limits)

## SDKs & Libraries
- **Official**: Python (`openai`), Node.js (`openai`)
- **Community**: Go, Rust, Java, C#, Ruby

## Use Cases
- Conversational AI and chatbots
- Code generation and review
- Content creation and editing
- Data analysis and extraction
- Image generation (DALL-E)
- Speech-to-text (Whisper)
