# Tabnine

## Overview
Tabnine is an AI code assistant focused on privacy and enterprise security. Offers both cloud and self-hosted options with code trained on permissive licenses.

## Key Features
- Code completion
- Full-function suggestions
- Natural language to code
- Unit test generation
- Code review assistance
- Private deployment options
- Zero data retention
