# Codeium

## Overview
Codeium provides free AI-powered code autocomplete and chat for individual developers. Trains on permissively licensed code only for enterprise peace of mind.

## Key Features
- Free unlimited autocomplete
- In-editor chat
- 70+ language support
- 40+ IDE support
- Natural language search
- Code explanation
- Enterprise deployment options
