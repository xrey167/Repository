# Midjourney

AI image generation via Discord.

## Access
- Discord bot (primary)
- Web: https://midjourney.com

## Commands
| Command | Description |
|---------|-------------|
| `/imagine` | Generate image |
| `/blend` | Blend images |
| `/describe` | Image to prompt |

## Prompt Tips
- Style: `--style raw`
- Aspect: `--ar 16:9`
- Quality: `--q 2`
- Version: `--v 6`

## Example
```
/imagine a cyberpunk city at night, neon lights --ar 16:9 --v 6
```
