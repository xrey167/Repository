# DeepSeek

## Overview
DeepSeek is a Chinese AI lab producing highly capable open-source models. Known for excellent coding performance and cost-effective API pricing.

## Key Features
- Open-source models
- Excellent coding capability
- Mixture of Experts architecture
- Very competitive pricing
- Long context support
- Reasoning capabilities (R1)

## Models

| Model | Description | Context | Best For |
|-------|-------------|---------|----------|
| deepseek-chat | General conversation | 64K | Chat, general use |
| deepseek-coder | Code specialized | 64K | Programming |
| deepseek-r1 | Reasoning model | 64K | Complex reasoning |
| deepseek-v3 | Latest flagship | 64K | All tasks |

## Pricing

| Model | Input | Output |
