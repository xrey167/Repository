# Claude Code

## Overview
Claude Code is Anthropic's official CLI tool for agentic coding with Claude. Provides terminal-based AI assistance for software development tasks with full codebase context.

## Key Features
- Terminal-based interface
- Full codebase awareness
- Multi-file editing
- Git integration
- Tool use (bash, file ops)
- Conversation memory
- MCP server support

## Products

| Feature | Description |
|---------|-------------|
| Chat | Conversational coding |
| Edit | AI-powered file editing |
| Bash | Command execution |
| Git | Version control integration |
| Plan Mode | Implementation planning |

## Installation

```bash
npm install -g @anthropic-ai/claude-code
```

## Configuration

```bash
# Set API key
export ANTHROPIC_API_KEY=your-key

# Run Claude Code
claude
```

## Key Commands
- `/help` - Show available commands
- `/clear` - Clear conversation
- `/compact` - Summarize context
- `/cost` - Show token usage
- `/model` - Switch models

## Use Cases
- Codebase exploration
