# Midjourney

## Overview
Midjourney is an AI image generation service known for artistic, high-quality outputs. Accessed primarily through Discord, with a web interface in development.

## Key Features
- Artistic image generation
- Style variations
- Image upscaling
- Inpainting/editing
- Aspect ratio control
- Style reference
- Character consistency

## Usage (Discord)

```
/imagine prompt: a cyberpunk city at night, neon lights --ar 16:9 --v 6.1

/describe [upload image]

/blend [image1] [image2]
```

## Parameters
- `--ar` - Aspect ratio (16:9, 1:1, 9:16)
- `--v` - Model version
- `--style` - Style preset
- `--chaos` - Variation amount
- `--stylize` - Artistic strength
- `--no` - Negative prompts
