# Cursor

## Overview
Cursor is an AI-first code editor built on VS Code. Features deep AI integration for editing, debugging, and understanding codebases with multiple model options.

## Key Features
- AI-powered code editing (Cmd+K)
- Codebase-aware chat
- Multi-file editing
- Tab completion
- Multiple AI model options
- VS Code extension compatibility
- Privacy mode

## Products

| Feature | Description |
|---------|-------------|
| Cursor Tab | Intelligent autocomplete |
| Cmd+K | Inline editing with AI |
| Chat | Codebase-aware conversations |
| Composer | Multi-file generation |

## Configuration
```json
{
  "cursor.general.aiProvider": "anthropic",
  "cursor.general.defaultModel": "claude-3.5-sonnet"
}
```
