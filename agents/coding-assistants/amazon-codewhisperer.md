# Amazon CodeWhisperer (Amazon Q Developer)

## Overview
Amazon Q Developer (formerly CodeWhisperer) is AWS's AI coding assistant. Deep integration with AWS services and security scanning capabilities.

## Key Features
- Code suggestions
- Security scanning
- AWS service integration
- Reference tracking (license)
- Infrastructure as Code support
- CLI completions
- Multiple language support

## Products

| Product | Description | Best For |
|---------|-------------|----------|
| Q Developer Free | Individual tier | Personal use |
| Q Developer Pro | Professional tier | Teams |
| Q Developer in IDE | Code assistance | Development |
| Q Developer in CLI | Terminal help | DevOps |

## Pricing

| Tier | Price | Includes |
|------|-------|----------|
| Individual | Free | Code suggestions, security scans |
| Professional | $19/user/month | + Admin, SSO, policies |
