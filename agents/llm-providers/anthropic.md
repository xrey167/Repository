# Anthropic

## Overview
Anthropic develops Claude, a family of AI assistants focused on being helpful, harmless, and honest. Claude excels at analysis, coding, writing, and complex reasoning tasks.

## Key Features
- Constitutional AI for safety
- Extended context windows (200K tokens)
- Strong coding and analysis capabilities
- Vision capabilities
- Tool use and function calling
- Computer use (beta)

## Models

| Model | Description | Context | Best For |
|-------|-------------|---------|----------|
| claude-opus-4-5 | Most capable model | 200K | Complex tasks, research |
| claude-sonnet-4 | Balanced performance | 200K | General use, coding |
| claude-3-5-haiku | Fast and efficient | 200K | Quick tasks, high volume |

## API Information
- **Base URL**: `https://api.anthropic.com/v1`
- **Authentication**: `x-api-key` header
- **Rate Limits**: Varies by tier

## SDKs & Libraries
- **Official**: Python (`anthropic`), TypeScript (`@anthropic-ai/sdk`)
- **Community**: Go, Rust, Java

## Use Cases
- Long document analysis
- Code generation and debugging
- Research and writing assistance
- Data extraction and processing
- Agentic workflows
- Computer automation
