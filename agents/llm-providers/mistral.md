# Mistral AI

## Overview
Mistral AI develops efficient, open-weight language models with strong performance-to-size ratios. Known for fast inference and competitive capabilities.

## Key Features
- Efficient architecture
- Open-weight models available
- Function calling support
- JSON mode
- Code-specialized models (Codestral)
- Mixture of Experts (MoE)

## Models

| Model | Description | Context | Best For |
|-------|-------------|---------|----------|
| mistral-large | Most capable | 128K | Complex tasks |
| mistral-medium | Balanced | 32K | General use |
| mistral-small | Fast and efficient | 32K | High-volume |
| codestral | Code-specialized | 32K | Programming |
| mixtral-8x22b | MoE flagship | 64K | Diverse tasks |
| mixtral-8x7b | Efficient MoE | 32K | Cost-effective |
