# GitHub Copilot

AI pair programmer by GitHub/Microsoft.

## Features
- **Inline**: Tab completions
- **Chat**: Cmd+I for inline, Cmd+Shift+I for panel
- **Workspace**: @workspace for codebase context

## Configuration
```json
// settings.json
{
  "github.copilot.enable": {
    "*": true,
    "markdown": false
  }
}
```

## VS Code Extensions
- GitHub Copilot
- GitHub Copilot Chat
