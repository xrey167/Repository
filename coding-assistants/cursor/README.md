# Cursor

AI-first code editor built on VS Code.

## Features
- **Tab**: Autocomplete with context
- **Cmd+K**: Edit selected code
- **Cmd+L**: Chat with codebase
- **Composer**: Multi-file edits

## Configuration
```
.cursor/
├── rules/           # Custom AI rules
└── .cursorrules     # Project-level rules
```

## Tips
- Use `@file` to reference files in chat
- Use `@codebase` for full context search
- Add rules for project conventions
