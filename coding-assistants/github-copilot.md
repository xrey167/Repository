# GitHub Copilot

## Overview
GitHub Copilot is an AI pair programmer that offers code suggestions directly in your editor. Powered by OpenAI Codex and GPT-4, integrated deeply with GitHub ecosystem.

## Key Features
- Real-time code completions
- Multi-line suggestions
- Chat interface (Copilot Chat)
- Pull request summaries
- Documentation generation
- Test generation
- CLI assistance

## Products

| Product | Description | Best For |
|---------|-------------|----------|
| Copilot Individual | Personal use | Individual developers |
| Copilot Business | Team features | Organizations |
| Copilot Enterprise | Custom models, knowledge bases | Large enterprises |
| Copilot Chat | Conversational coding | Interactive help |

```json

  "github.copilot.enable": {
    "*": true,
    "markdown": false
  }
}
```
